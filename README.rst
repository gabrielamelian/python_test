delete_things
=============

A collection of scripts to fulfill interview requirements

Usage
-----

* Answers to the questions from the interview can be found on ANSWERS.md
* Script to sort the contents of a file can be found on delete_things/sort_file
* Script to delete files and directories can be found on delete_things/delete
